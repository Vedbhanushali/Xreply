# Creating Extension Icons

The Chrome extension requires three icon files. You have several options:

## Option 1: Use Online Icon Generators

1. Visit [Favicon.io](https://favicon.io/) or [RealFaviconGenerator](https://realfavicongenerator.net/)
2. Upload an image or create a simple icon
3. Download the generated icons
4. Rename them to:
   - `icon16.png` (16x16 pixels)
   - `icon48.png` (48x48 pixels)
   - `icon128.png` (128x128 pixels)

## Option 2: Create Simple Icons

You can create simple icons using any image editor:

- **Size**: 16x16, 48x48, and 128x128 pixels
- **Format**: PNG with transparency
- **Design**: Simple icon representing "reply" or "AI" (e.g., speech bubble, sparkles, etc.)

## Option 3: Use Placeholder (Temporary)

For testing, you can create simple colored squares:

1. Use any image editor
2. Create three square images with the specified sizes
3. Fill with a gradient color (e.g., purple/blue to match the button)
4. Add text "X" or "R" if desired

## Option 4: Download from Icon Libraries

- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)
- [Font Awesome](https://fontawesome.com/icons)

Search for "reply" or "chat" icons and download in PNG format.

## Quick Test Without Icons

The extension will work without icons, but Chrome will show a warning. You can still test all functionality!
